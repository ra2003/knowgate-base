# KnowGate Base Libraries

These are foundation libraries compatible with Java 1.6. They include:

## Debug
- Debug traces logger

## Arrays and String
- Array helper methods
- Tuples (Pair and Triplet)
- String helper methods

## IO
- File utils
- Character set inference
- Stream to String bridging
- Stream to Stream pipes

## Localisation
- Currency helper methods
- Date helper methods
- GIS interfaces

## Security
- Encryption MD5 and RC4.
- Global unique identifier generation

## System
- Shell
